# Via Sacra Digital

Aplicativo católico digital com orações, devocionais, Bíblia, fórum e recursos espirituais.

## ✨ Funcionalidades

- 📖 **Bíblia Sagrada**: Leitura, busca de versículos e navegação bíblica
- 🙏 **Orações**: Biblioteca de orações católicas tradicionais
- 📅 **Devocionais**: Entradas diárias de reflexão espiritual
- 💬 **Fórum**: Comunidade para discussões espirituais
- 🎵 **Música**: Cantos e músicas católicas
- ⛪ **Paróquias**: Informações sobre paróquias locais
- 🤖 **IA Católica**: Assistente espiritual com IA
- 📿 **Rosário**: Guia para rezar o rosário
- 💝 **Doações**: Sistema de contribuições

## 🌟 Versículo do Dia

O aplicativo inclui uma funcionalidade especial de **Versículo do Dia** inspirada no site [bibliaon.com](https://www.bibliaon.com):

- **Versículo único para cada dia**: Baseado no dia do ano, garantindo variedade
- **Coleção de 50+ versículos inspiradores**: Selecionados para edificação espiritual
- **Narração por voz**: Funcionalidade de texto-para-fala em português brasileiro
- **Busca inteligente**: Sistema de busca por palavras-chave ou referências bíblicas

### 📚 Versículos Incluídos

- João 3:16 - O amor de Deus
- Filipenses 4:13 - Força em Cristo
- Salmo 23:1 - O Senhor é meu pastor
- Isaías 41:10 - Não temas, eu estou contigo
- Mateus 11:28 - Vinde a mim, todos os cansados
- Lucas 1:37 - Para Deus nada é impossível
- E muitos outros versículos edificantes...

## 🚀 Instalação

1. **Instalar dependências:**
   ```bash
   npm install
   ```

2. **Executar em modo de desenvolvimento:**
   ```bash
   npm run dev
   ```

3. **Abrir no navegador:**
   O aplicativo será aberto automaticamente em `http://localhost:3000`

## 📱 Scripts Disponíveis

- `npm run dev` - Executa em modo de desenvolvimento
- `npm run build` - Cria build de produção
- `npm run preview` - Visualiza build de produção

## 🛠️ Tecnologias

- **Frontend**: React 18 + Vite
- **Estilização**: Tailwind CSS
- **Roteamento**: React Router DOM
- **Ícones**: Lucide React
- **Utilitários**: date-fns
- **Funcionalidades**: Text-to-Speech, Busca Inteligente

## 🙏 **Santo Rosário Completo**

O aplicativo agora inclui o **Santo Rosário completo** baseado no [Devocionário Santo Rosário](https://evangelizandocommaria.com.br/livros/Devocion%C3%A1rio%20Santo%20Ros%C3%A1rio.pdf):

### ✨ **Funcionalidades do Rosário:**

- **Mistérios Gozosos**: Segunda-feira, Quinta-feira e Sábado
- **Mistérios Dolorosos**: Terça-feira e Sexta-feira  
- **Mistérios Gloriosos**: Quarta-feira e Domingo
- **Orações em Latim e Português**: Texto completo de todas as orações
- **Meditações Bíblicas**: Reflexões para cada mistério
- **Sistema de Décadas**: Contador interativo para acompanhar a oração
- **Narração por Voz**: Funcionalidade de áudio para todas as orações

### 📖 **Orações Incluídas:**

- **Sinal da Cruz** (latim e português)
- **Credo** (latim e português)
- **Pai Nosso** (latim e português)
- **Ave Maria** (latim e português)
- **Glória ao Pai** (latim e português)
- **Oração de Fátima** (latim e português)
- **Salve Rainha** (latim e português)
- **Terço da Divina Misericórdia** (completo)
- **Terço de São Miguel Arcanjo** (completo)
- **Consagração ao Imaculado Coração de Maria**
- **Consagração ao Sagrado Coração de Jesus**

### 🎯 **Como Usar:**

1. **Acesse a página "Rosário"** no menu lateral
2. **Escolha o tipo de mistério** (Gozosos, Dolorosos, Gloriosos)
3. **Navegue pelos mistérios** usando os botões de navegação
4. **Acompanhe as décadas** com o contador visual
5. **Medite** com as reflexões bíblicas
6. **Clique em "Ler Completo"** para ver a oração expandida
7. **Veja primeiro em português** e clique em "Ver em Latim" se quiser
8. **Ouça as orações** usando a funcionalidade de áudio

### 📚 **Base Teológica:**

Todo o conteúdo está baseado no devocionário oficial e segue a tradição católica, incluindo:
- Referências bíblicas precisas para cada mistério
- Textos em latim conforme a tradição litúrgica
- Traduções em português brasileiro
- Meditações baseadas na doutrina católica

## 🤖 IA Católica com Google Gemini

O aplicativo agora inclui **Inteligência Artificial real** integrada com a API do Google Gemini:

### ✨ **Funcionalidades da IA:**

- **Assistente Espiritual Inteligente**: Respostas baseadas na doutrina católica
- **Geração de Orações**: Orações personalizadas para diferentes situações
- **Análise Bíblica**: Explicações e insights sobre versículos
- **Orientação Espiritual**: Conselhos práticos para a vida cristã
- **Orações para Santos**: Orações específicas baseadas na vida dos santos

### 🔧 **Tecnologia:**

- **API**: Google Gemini 2.0 Flash
- **Modelo**: `gemini-2.0-flash`
- **Idioma**: Português brasileiro
- **Base**: Doutrina católica tradicional
- **Prompting**: Prompts especializados para cada funcionalidade

### 📱 **Como Usar:**

1. **Acesse a página "IA Católica"**
2. **Escolha uma categoria**: Geral, Orações, Bíblia ou Orientação
3. **Faça sua pergunta** no chat
4. **Receba respostas inteligentes** em tempo real

### 🎯 **Exemplos de Uso:**

- **Orações**: "Gere uma oração para agradecer"
- **Bíblia**: "Explique João 3:16"
- **Orientação**: "Como crescer na fé?"
- **Geral**: "O que significa a Eucaristia?"

### 🛡️ **Segurança e Confiabilidade:**

- **Fallback automático** em caso de erro da API
- **Validação de respostas** para garantir qualidade
- **Prompts especializados** para cada tipo de pergunta
- **Base doutrinária** católica confiável

## Orações e Devoções

A página de Orações e Devoções oferece uma coleção abrangente de orações católicas tradicionais e especiais do Opus Dei.

### Funcionalidades

- **Categorias Organizadas**: Orações tradicionais, devocionais, diárias, do rosário, novenas e ladainhas
- **Busca Inteligente**: Pesquise por nome ou conteúdo das orações
- **Áudio Narração**: Ouça as orações com síntese de voz em português ou latim
- **Versões Bilíngues**: Todas as orações principais disponíveis em português e latim
- **Interface Intuitiva**: Botão para alternar entre idiomas e indicador visual do idioma atual

### Orações do Opus Dei

A aplicação inclui uma seleção especial de orações inspiradas na espiritualidade do Opus Dei, incluindo:

- **Orações Tradicionais**: Pai Nosso, Ave Maria, Credo, Glória, Salve Rainha
- **Orações Devocionais**: 
  - Oração de São Josemaria
  - Oração pela Santificação do Trabalho
  - Oração pela Família
  - Oração pelos Doentes
  - Oração pelos Defuntos
  - Oração pela Igreja
  - Oração pela Paz
  - Oração de Ação de Graças
  - Oração de Contrição
  - Oração pela Vocação

### Como Usar

1. **Navegue pelas Categorias**: Use as abas para encontrar orações por tipo
2. **Busque Orações**: Use a barra de busca para encontrar orações específicas
3. **Alterne Idiomas**: Clique no botão de idiomas para ver versões em português ou latim
4. **Ouça as Orações**: Use o botão de áudio para narração automática
5. **Fonte das Orações**: Todas as orações do Opus Dei são claramente identificadas

### Características Técnicas

- **Síntese de Voz**: Suporte para português brasileiro e latim
- **Responsivo**: Interface adaptada para dispositivos móveis e desktop
- **Acessibilidade**: Controles intuitivos e indicadores visuais claros
- **Performance**: Carregamento otimizado e busca em tempo real
